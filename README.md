# zachet
Промежуточная аттестация

ER в формате pdf находится в папке ERD.

Установка зависимостей:

    pip install -r requirements.txt

Внести изменения в database/config.py, подставив свои значения:

    'user': '<user>',
    
    'password': '<password>',
    
    'host': '<host>',
    
    'port': '<port>'
